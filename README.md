## GitHub Pages 发布

这个版本是纯静态网站，无需构建。

1. 把 `index.html`、`styles.css`、`script.js` 上传到仓库根目录
2. 打开 GitHub 仓库 `Settings` → `Pages`
3. 在 `Build and deployment` 中选择 `Deploy from a branch`
4. Branch 选择 `main`，Folder 选择 `/ (root)`
5. 保存后等待 GitHub Pages 发布

如果你的仓库名是 `ich-natur-zeitraum`，发布地址通常会是：

`https://tina10177.github.io/ich-natur-zeitraum/`
